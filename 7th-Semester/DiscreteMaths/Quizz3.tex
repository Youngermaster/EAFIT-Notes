Sean A, B, C conjuntos, probar que:

\[ (A - C) \cap (C - B) = \emptyset \]

\[ (A \cap C^c) \cap (C \cap B^c) = \emptyset \]

\[ ((A^c)^c \cap C^c) \cap ((C^c)^c \cap B^c) = \emptyset \]

\[ (A^c \cup C)^c \cap (C^c \cup B)^c = \emptyset \]

\[ (C \cup A^c)^c \cap (C^c \cup B)^c = \emptyset \]

\[ C \cap C^c = \emptyset \]

\[ \emptyset = \emptyset \]
